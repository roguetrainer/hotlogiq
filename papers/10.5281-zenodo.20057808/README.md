# The Fano-SYK Model: Bruhat-Tits Buildings, Non-Associative Fermionic Couplings, and the Geometric Impedance of Pre-Thermal Scrambling

**Paper:** 267 | **Portfolio A** — Core Engine

**DOI:** [10.5281/zenodo.20057808](https://zenodo.org/records/20057808)

## Abstract

Restricts the SYK model's random couplings to Fano-line co-associative quadruples. The resulting Fano-SYK model exhibits geometric impedance of pre-thermal scrambling via Bruhat-Tits building geometry.
